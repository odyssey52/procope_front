# 프로코프(procope)

애자일 팀 원픽 회고 관리 서비스입니다. 목표관리와 회고를 수치화하여 관리할 수 있게 도와드립니다. 회고 진행사항을 한눈에 확인해 보세요.

## introduce

**프로코프(procope)** 의 프론트엔드 코드 저장소 입니다.

## tech stack

- React
- TypeScript
- Tanstack query
- Zustand
